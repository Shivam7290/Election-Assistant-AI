## 🌟 Elevator Pitch

Navigating the democratic process can be intimidating, especially for first-time voters. **VoteGuide AI** bridges this gap by providing an accessible, bilingual, and highly interactive web platform. Powered by Google's Gemini 2.5 Flash AI, it delivers strictly neutral, factual, and easy-to-understand guidance on voting timelines, registration, and required documents.

---

## ✨ Core Features

- **🤖 Floating AI Assistant** — A sleek chat widget powered by Gemini 2.5 Flash. Strictly prompted to avoid bias and provide step-by-step assistance in English and Hindi.
- **🌗 Premium UI & Dark Mode** — Modern glassmorphism design with seamless light/dark theme switching that remembers user preferences.
- **✨ Micro-interactions & Animations** — Built with `framer-motion` for smooth animations and confetti celebrations for positive civic actions.
- **🔍 Voter ID Verification** — Interactive UI flow to demonstrate instant electoral status verification.
- **🌐 Bilingual Support** — English/Hindi toggle that translates both static UI and AI system prompts dynamically.
- **♿ Accessibility** — ARIA labels, semantic HTML, skip navigation, and screen reader support throughout.
